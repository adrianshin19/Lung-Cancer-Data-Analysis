# Lung Cancer Data Analysis

This project explores a publicly available lung cancer dataset to investigate relationships between clinical variables and outcomes.

## Project Components

- **Python (Jupyter Notebook)**  
  - Data preprocessing  
  - Linear regression modeling  
  - Basic statistical evaluation  

- **R Script**
  - Data visualization
  - Exploratory plots of key variables

## Tools Used
- Python (Regression)
- R Shiny Teal (Visualization)

## Notes
The Python part of the project was completed as part of a mentored group research experience, where I contributed to the regression modeling and data analysis components, while the R project was an individual project to learn R Shiny Teal.
